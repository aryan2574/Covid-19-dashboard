# Covid-19-dashboard
Covid-19 dashboard created using React + TS.

### Api Used : https://data.covid19india.org/v4/min/data.min.json

## How to run
1. Clone the repo
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:3000/` in your browser
5. Check Covid 19 data for your state
